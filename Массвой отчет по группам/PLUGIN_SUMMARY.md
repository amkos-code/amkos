# Сводка по плагину "Общий отчет по группам"

## 🎯 Основные функции

### 1. Загрузка CSV файла
- Пользователь загружает CSV файл со списком групп
- Формат: столбец A, каждая группа с новой строки
- Автоматическое чтение и обработка файла

### 2. Поиск групп в Moodle
- Автоматический поиск групп по названиям из CSV
- Извлечение информации о курсах
- Получение списка участников

### 3. Генерация отчета
- Сбор данных по всем найденным группам
- Извлечение оценок за тесты (quiz и lesson модули)
- Формирование единой таблицы

### 4. Экспорт в Excel
- Генерация XLS файла через MoodleExcelWorkbook
- Красивое форматирование (заголовки, границы, цвета)
- Автоматическая загрузка файла пользователем

## 🔧 Технические детали

### Расположение в меню
**Администрирование** → **Оценки** → **Общий отчет по группам**

### Права доступа
- `local/groupreport:view` - просмотр отчета
- `local/groupreport:manage` - управление отчетом

Роли по умолчанию:
- Manager: CAP_ALLOW (оба права)
- Admin: CAP_ALLOW (manage)
- Editingteacher: CAP_ALLOW (view)
- Teacher: CAP_ALLOW (view)

### Структура данных отчета
```php
[
    'courses' => [courseid => fullname],
    'groups' => [groupid => groupname],
    'students' => [userid => fullname],
    'rows' => [
        ['course' => '', 'group' => '', 'student' => '', 'grade' => '']
    ]
]
```

### Поддерживаемые типы активностей
1. **Quiz** - тесты Moodle
2. **Lesson** - уроки с вопросами

## 📊 Рабочий процесс

```
1. Пользователь загружает CSV
   ↓
2. Чтение названий групп
   ↓
3. Поиск групп в БД
   ↓
4. Для каждой группы:
   - Получение курса
   - Получение участников
   - Для каждого участника:
     * Поиск тестов
     * Получение оценок
   ↓
5. Формирование отчета
   ↓
6. Предварительный просмотр
   ↓
7. Экспорт в Excel (по запросу)
```

## 🚀 Установка

1. Скопировать папку `local/groupreport` в Moodle
2. Администрирование → Уведомления → Обновить БД
3. Готово! Плагин в меню "Оценки"
