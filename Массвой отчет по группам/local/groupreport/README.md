# Общий отчет по группам (local_groupreport)

Плагин для Moodle 4.5, который позволяет создавать сводные отчеты по нескольким группам одновременно.

## Возможности

- Загрузка списка групп из CSV файла
- Автоматический поиск групп в системе Moodle
- Формирование единого отчета по всем выбранным группам
- Отображение: наименования курса, ФИО участников, оценки за тесты
- Экспорт отчета в формате Excel (.xls)

## Установка

1. Скопируйте папку `groupreport` в директорию `/local/` вашего Moodle
2. Войдите в систему как администратор
3. Перейдите в "Администрирование" -> "Уведомления" для завершения установки
4. Плагин появится в разделе "Администрирование" -> "Оценки" -> "Общий отчет по группам"

## Использование

1. Подготовьте CSV файл со списком групп
   - В столбце A (первый столбец) укажите названия групп
   - Каждая группа должна быть с новой строки
   - Пример:
     ```
     Группа 1
     Группа 2
     Группа 3
     ```

2. Загрузите CSV файл через интерфейс плагина

3. Система автоматически найдет все группы и сформирует отчет

4. Просмотрите предварительный просмотр отчета

5. Нажмите кнопку "Экспорт в Excel" для скачивания готового отчета

## Структура отчета

Отчет содержит следующие колонки:
- Наименование курса
- Группа
- ФИО участника
- Оценка за тест

## Требования

- Moodle 4.5 или выше
- PHP 7.4 или выше
- Библиотека Spreadsheet (входит в состав Moodle)

## Права доступа

По умолчанию права доступа имеют:
- Менеджеры (manager) - полный доступ
- Администраторы (admin) - полный доступ
- Редактирующие преподаватели (editingteacher) - просмотр
- Преподаватели (teacher) - просмотр

## Расположение в меню

Плагин добавляет пункт меню в раздел оценок:
**Администрирование** → **Оценки** → **Общий отчет по группам**
