# Быстрый старт

## 📦 Установка за 3 шага

1. **Скопируйте папку** `local/groupreport` в корень вашего Moodle
2. **Откройте Moodle** как администратор → Администрирование → Уведомления → Обновить БД
3. **Готово!** Плагин появится в меню "Администрирование" → "Оценки"

## 🎯 Первое использование

### Шаг 1: Подготовьте CSV
Создайте файл `groups.csv` с содержимым:
```
Ваша Группа 1
Ваша Группа 2
Ваша Группа 3
```

### Шаг 2: Загрузите CSV
- Перейдите в: **Администрирование** → **Оценки** → **Общий отчет по группам**
- Выберите файл → Загрузите ваш CSV
- Нажмите "Загрузить и создать отчет"

### Шаг 3: Получите отчет
- Просмотрите предварительный просмотр
- Нажмите "Экспорт в Excel"
- Готово! Файл загружен

## ✅ Проверка работы

После установки проверьте:
- ✅ Плагин виден в меню "Оценки"
- ✅ Можно загрузить CSV файл
- ✅ Генерируется предварительный просмотр
- ✅ Работает экспорт в Excel

## 🔍 Если что-то не так

1. **Не видно в меню?** → Очистите кеш Moodle
2. **Ошибка при загрузке?** → Проверьте права на файлы
3. **Пустой отчет?** → Убедитесь, что группы существуют
