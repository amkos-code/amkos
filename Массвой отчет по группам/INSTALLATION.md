# Инструкция по установке плагина "Общий отчет по группам"

## Шаг 1: Размещение файлов плагина

Скопируйте папку `groupreport` в директорию `/local/` вашей установки Moodle.

Струкка должна выглядеть так:
```
moodle/
└── local/
    └── groupreport/
        ├── db/
        │   ├── access.php
        │   └── install.xml
        ├── lang/
        │   └── ru/
        │       └── local_groupreport.php
        ├── index.php
        ├── report.php
        ├── settings.php
        ├── version.php
        └── README.md
```

## Шаг 2: Установка через веб-интерфейс

1. Войдите в Moodle как администратор
2. Перейдите в: **Администрирование** → **Уведомления**
3. Система автоматически обнаружит новый плагин
4. Нажмите кнопку "Обновить базу данных Moodle сейчас"
5. Дождитесь завершения установки

## Шаг 3: Проверка установки

После установки убедитесь, что плагин появился в меню:
- Перейдите в: **Администрирование** → **Оценки**
- Вы должны увидеть пункт "**Общий отчет по группам**"

## Шаг 4: Настройка прав доступа (опционально)

По умолчанию права доступа настроены следующим образом:

- **Менеджеры** и **Администраторы**: полный доступ
- **Редактирующие преподаватели** и **Преподаватели**: просмотр

Для изменения прав доступа:
1. Перейдите в: **Администрирование** → **Пользователи** → **Права доступа** → **Определение ролей**
2. Выберите нужную роль
3. Найдите способности:
   - `local/groupreport:view` - просмотр отчета
   - `local/groupreport:manage` - управление отчетом
4. Установите нужные разрешения

## Шаг 5: Использование плагина

### Подготовка CSV файла

Создайте CSV файл со списком групп:
- Откройте Excel или любой текстовый редактор
- В столбец A (первый столбец) запишите названия групп
- Каждая группа должна быть с новой строки
- Сохраните файл в формате CSV

Пример содержимого файла `groups.csv`:
```
Группа 1
Группа 2
Группа 3
Группа 4
```

### Генерация отчета

1. Перейдите в: **Администрирование** → **Оценки** → **Общий отчет по группам**
2. Нажмите кнопку "Выберите файл" и загрузите ваш CSV файл
3. Нажмите "Загрузить и создать отчет"
4. Система найдет все группы и сформирует отчет

### Просмотр и экспорт

После генерации отчета вы увидите:
- Краткую сводку (количество курсов, групп, студентов)
- Кнопку "Экспорт в Excel"
- Таблицу предварительного просмотра

Для скачивания отчета:
1. Нажмите кнопку "**Экспорт в Excel**"
2. Файл автоматически загрузится на ваше устройство
3. Откройте файл в Excel или совместимой программе

## Возможные проблемы и их решение

### Плагин не отображается в меню

1. Убедитесь, что файлы скопированы в правильную директорию
2. Проверьте права доступа к файлам
3. Очистите кеш Moodle: **Администрирование** → **Разработка** → **Очистить кеш**
4. Убедитесь, что у вашей роли есть необходимые права доступа

### Ошибка при загрузке CSV

1. Убедитесь, что файл имеет расширение `.csv`
2. Проверьте кодировку файла (рекомендуется UTF-8)
3. Убедитесь, что названия групп указаны точно (без лишних пробелов)
4. Проверьте, что группы существуют в Moodle

### В отчете отсутствуют данные

1. Убедитесь, что в указанных группах есть студенты
2. Проверьте, что в курсах есть тесты с оценками
3. Убедитесь, что у студентов есть оценки за тесты

### Проблемы с экспортом в Excel

1. Проверьте настройки безопасности браузера
2. Убедитесь, что браузер разрешает загрузку файлов
3. Попробуйте использовать другой браузер

## Технические требования

- **Moodle**: 4.5 или выше
- **PHP**: 7.4 или выше
- **База данных**: MySQL, PostgreSQL, MariaDB
- **Библиотеки**: Spreadsheet (входит в состав Moodle)

## Поддержка

При возникновении проблем обратитесь к системному администратору вашего Moodle.
